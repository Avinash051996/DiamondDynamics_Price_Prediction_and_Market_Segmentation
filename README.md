## 🚀 Quick Start

### 1. Add the Dataset
Download the Diamonds CSV (~54k rows) and save to:
```
diamond_dynamics/data/diamonds.csv
```

> ⚡ **No dataset?** `pipeline.py` will automatically generate a realistic  
> synthetic dataset (~54k rows) for testing/demo purposes.

### 2. Run the Training Pipeline
```bash
python pipeline.py
```
This will:
- Clean & preprocess data
- Handle outliers (IQR) and skewness (log transform)
- Engineer features (volume, dimension_ratio, etc.)
- Train **5 ML regression models** + **NumPy ANN**
- Evaluate all models (MAE, RMSE, R²)
- Train **K-Means clustering** with optimal K
- Save all models to `models/`
- Save all EDA & comparison plots to `plots/`

### 3. Launch the Streamlit App
```bash
streamlit run streamlit_app/app.py
```
Open your browser at **http://localhost:8501**

## 📊 Dataset — Column Descriptions

| Column   | Description |
|----------|-------------|
| carat    | Weight (carats). Primary price driver. |
| cut      | Cut quality: Fair → Good → Very Good → Premium → Ideal |
| color    | Color: D (best) → J (worst) |
| clarity  | Clarity: IF → VVS1 → VVS2 → VS1 → VS2 → SI1 → SI2 → I1 |
| depth    | Total depth % = z / mean(x,y) × 100 |
| table    | Width of top facet as % of diameter |
| price    | Price in USD (target variable) |
| x, y, z  | Length / Width / Height in mm |

---

## 🔬 ML Pipeline Details

### Preprocessing
- Zero/negative x, y, z → replaced with median
- IQR-based outlier removal on `carat, price, x, y, z`
- Log1p transform on right-skewed columns

### Feature Engineering
| Feature | Formula |
|---------|---------|
| volume | x × y × z |
| dimension_ratio | (x + y) / (2 × z) |
| price_inr | price × 83.5 |
| carat_category | Light / Medium / Heavy |

### Encoding
- **Ordinal Encoding** for cut, color, clarity (preserves natural order)

### Regression Models

| Model | Notes |
|-------|-------|
| Linear Regression | Baseline |
| Ridge Regression | L2 regularisation (α=10) |
| Decision Tree | max_depth=12 |
| Random Forest | 150 trees, max_depth=14 |
| KNN Regressor | k=7, distance-weighted |
| ANN (NumPy) | 128→64→32→1, Adam, 60 epochs |

Best model (highest R²) is saved as `best_regression_model.pkl`.

### Clustering
- **K-Means** with Elbow + Silhouette score to find optimal K
- PCA 2D projection for visualisation
- Clusters auto-labelled by ascending mean carat:
  - 🟢 Affordable Small Diamonds
  - 🟡 Mid-range Balanced Diamonds
  - 💎 Premium Heavy Diamonds

## 📝 Notes
- USD → INR conversion rate is fixed at **₹83.5** — update `USD_TO_INR` in `pipeline.py` and `app.py` as needed.
- The ANN is built entirely in NumPy (no TensorFlow/PyTorch required) for portability.
- To swap in XGBoost or LightGBM, simply add them to the `models` dict in `build_regression_models()`.
